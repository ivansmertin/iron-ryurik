# Designer Handoff: Open Questions Checklist

- [ ] Какой уровень акцентности у `hybrid` стилистики считаем целевым (низкий/средний/высокий)?
- [ ] Нужен ли отдельный визуальный язык для каждой роли или единая база с малыми отличиями?
- [ ] Какая стратегия для плотных таблиц на mobile: горизонтальный скролл или карточная трансформация?
- [ ] Нужны ли иллюстративные empty states или только типографические/иконные?
- [ ] Какой формат передачи предпочтителен для дизайн-токенов (именование, шкалы, семантика)?
- [ ] Приоритет dark/light: равный или dark-first?
