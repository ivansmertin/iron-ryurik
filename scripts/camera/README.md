### Настройка скрипта аналитики для Tapo

Этот скрипт использует нейросеть (YOLOv8) для поиска людей в кадре с RTSP-потока камеры Tapo и отправляет их количество в ваше веб-приложение `Iron Rurik`.

#### 1. Подготовка камеры Tapo
1. Откройте приложение Tapo на телефоне.
2. Перейдите в настройки камеры -> `Расширенные настройки` (Advanced Settings) -> `Учетная запись камеры` (Camera Account).
3. Создайте имя пользователя и пароль.
4. Узнайте IP-адрес камеры в локальной сети (можно посмотреть в настройках роутера или в приложении Tapo `значок шестеренки` -> `Сведения о камере` -> `IP-адрес`).

#### 2. Установка Python и библиотек
Вам понадобится компьютер, который будет находиться в той же Wi-Fi сети, что и камера.
1. Установите Python (с официального сайта python.org).
2. Откройте консоль (терминал/командную строку) и установите библиотеки:
   ```bash
   pip install ultralytics opencv-python requests
   ```

#### 3. Настройка скрипта
Откройте скрипт `tracker.py` и измените настройки в самом верху файла:
- `RTSP_URL`: Вставьте логин, пароль и IP-адрес вашей камеры.
- `WEBHOOK_URL`: Замените на URL-адрес вашего развернутого приложения (если тестируете локально, оставьте `http://localhost:3000/api/webhooks/turnstile`).
- `WEBHOOK_SECRET`: Секретный ключ (должен совпадать с `TURNSTILE_SECRET` из `.env` приложения).

#### 4. Запуск
```bash
python tracker.py
```
Скрипт будет каждые N секунд брать кадр с камеры, считать на нём людей и отправлять данные в приложение! При работе откроется небольшое окошко видео, чтобы вы могли убедиться, что ИИ правильно распознаёт людей.
